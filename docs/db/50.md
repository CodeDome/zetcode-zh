## Perl DBI

Perl _DBI_ （数据库接口）是用于 Perl 编程语言的数据库访问模块。 它定义了一组提供标准数据库接口的方法，变量和约定。

其他语言也创建了用于数据库的类似通用接口。 Java 具有 JDBC，PHP PDO。
